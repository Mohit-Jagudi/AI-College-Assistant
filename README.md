# AI College Helpdesk Chatbot

A smart and interactive AI-powered chatbot-based college helpdesk project built as a single HTML file with embedded CSS, JavaScript, and API integration.  
This project is designed to help students get quick answers about college-related information in a simple and user-friendly way.  
The webpage includes a complete college website layout with sections like About, Courses, Admissions, Fees, Events, Placements, Faculty, Notices, and Contact, along with a chatbot interface.

## Live Demo

https://mohit-jagudi.github.io/AI-College-Assistant/

## Problem Statement

Design and develop an AI-powered chatbot that acts as a College Helpdesk Assistant.

The bot should be able to:
- Answer student queries  
- Provide college-related information  
- Assist with common tasks like:
  - Courses  
  - Fee structure  
  - Upcoming events  
  - Contact information

## How to Run

1. Download or clone the project  
2. Open the project folder  
3. Open the file: index.html
4. Run it in any web browser  
5. After opening in browser, a popup will appear asking for an API key  
 - Enter the API key   
 - Without API key, the helpdesk chatbot will not function  
6. After entering the API key, start interacting with the chatbot

---

## Working

1. User opens the website  
2. User enters a query in chatbot  
3. Query is processed through API  
4. AI generates response  
5. Response is displayed on screen
